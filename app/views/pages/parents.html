Parents

What/who is this site for?

We aim to provide high-quality lessons that teach children basic computer science and computational thinking by using the popular graphical programming language Scratch.

Originally, this site was created for a program at the Stamford Town Library. However, anyone is welcome to utilize these lessons, whether for their own children or for another class/program. If you do plan on using it for another class/program, we'd appreciate it if you could let us know.

What age children should take this course?

Originally, this course was intended for 3rd to 6th graders. However, the lessons should be suitable for younger ages, with adult help, or even older kids. More resources for older kids can be found here.

Can the lessons be done independently by the students?

The lessons are designed so that all concepts/instructions are clearly explained; however, there are many open-ended questions/projects, so it is advised that an adult be available to help the child, if needed. Parents are encouraged to follow along with their children in the lessons.
